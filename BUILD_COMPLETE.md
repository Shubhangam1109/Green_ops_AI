## 🎯 Features Implemented

### Dashboard (`/`)
- 4 KPI Cards: Total Carbon, Cloud Cost, Prediction, Green Score
- Line Chart: 5-month carbon trend
- Pie Chart: Service breakdown (EC2, Storage, Database, Network)
- AI Insights Widget: Actionable recommendations
- Top Recommendations Preview

### Forecast (`/forecast`)
- 6-month carbon emission forecast
- ML model prediction with confidence score
- Historical trend visualization
- Forecast data table
- Trend analysis insights

### Recommendations (`/recommendations`)
- All sustainability recommendations
- Priority levels (High/Medium/Low)
- Impact metrics (savings, carbon reduction)
- Implementation priority
- 4-step guidance for implementation

### Green Score (`/green-score`)
- Overall green score (A-F rating)
- 4-factor breakdown:
  - Carbon Intensity
  - Resource Efficiency
  - Unused Infrastructure
  - Storage Optimization
- Progress visualization
- Score scale reference
- Improvement tips

## 🚀 How to Run

### Step 1: Install Frontend Dependencies
```bash
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

**Access Dashboard**: http://localhost:3000

### Step 3 (Optional): Start Python ML Service
```bash
cd python-service
pip install -r requirements.txt
python train_model.py
python main.py
```

**FastAPI Docs**: http://localhost:8000/docs
